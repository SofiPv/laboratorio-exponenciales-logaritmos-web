# Laboratorio de Exponenciales y Logaritmos Web

Aplicación web didáctica multipágina para graficar funciones exponenciales y logarítmicas, analizar dominio, rango, asíntotas, transformaciones y resolver derivadas logarítmicas paso a paso.

---

## Objetivo del proyecto

Crear una herramienta de estudio para conectar temas de precálculo y cálculo diferencial.

El sistema permite:

- Graficar funciones exponenciales.
- Graficar funciones logarítmicas.
- Analizar dominio y rango.
- Identificar asíntotas horizontales y verticales.
- Aplicar transformaciones de funciones.
- Comparar funciones base con funciones transformadas.
- Resolver derivadas logarítmicas.
- Aplicar regla de la cadena.
- Aplicar derivación logarítmica.
- Resolver un modo examen.

## Módulos

### Funciones exponenciales

Trabaja funciones de la forma:

```text
f(x) = A · a^(b(x-h)) + k
```

Muestra:

- Dominio
- Rango
- Asíntota horizontal
- Crecimiento o decrecimiento
- Intersecciones
- Tabla de valores
- Gráfica

---

### Funciones logarítmicas

Trabaja funciones de la forma:

```text
f(x) = A · log_a(b(x-h)) + k
```

Muestra:

- Dominio
- Rango
- Asíntota vertical
- Crecimiento o decrecimiento
- Intersecciones
- Tabla de valores
- Gráfica

---

### Transformaciones

Incluye ejemplos de:

- Traslación vertical
- Traslación horizontal
- Reflexión respecto al eje x
- Reflexión respecto al eje y
- Dilatación vertical
- Cambio de asíntotas

---

### Derivadas logarítmicas

Incluye ejercicios como:

```text
d/dx ln(x)
d/dx log_a(x)
d/dx ln(x²+1)
d/dx ln(√(3-x²))
derivación logarítmica de y=x^x
```

---

### Modo examen

Incluye preguntas de opción múltiple con retroalimentación sobre:

- Dominio
- Rango
- Asíntotas
- Logaritmos
- Derivadas logarítmicas
- Derivación logarítmica

## Cómo usar el proyecto

Abre el archivo:

```text
index.html
```

en cualquier navegador moderno.

No requiere instalación de dependencias locales.

---

## Publicación en GitHub Pages

Este proyecto puede publicarse como sitio estático desde GitHub Pages.

Pasos generales:

1. Subir los archivos al repositorio.
2. Entrar a **Settings**.
3. Abrir **Pages**.
4. Seleccionar la rama `main`.
5. Guardar.
6. Abrir el enlace publicado por GitHub.

---

## Enfoque académico

El proyecto convierte material de funciones exponenciales, logarítmicas y derivadas logarítmicas en una experiencia interactiva.

La separación en páginas permite que cada bloque tenga su propio flujo de trabajo y no quede concentrado en una sola pantalla.
